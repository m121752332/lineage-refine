## 2. 視覺風格規格

### 背景設計

| 元素 | 來源 | 說明 |
|------|------|------|
| 背景圖 | `lineage_jpg/lineage_login.gif` | 動態 GIF，天堂城堡場景 |
| 背景疊層 | `rgba(0,0,0,0.55)` 半透明遮罩 | 確保登入框可讀性 |
| 粒子效果 | CSS + JS 動態生成 | 模擬燭光火星飄浮感 |

### 燭光粒子規格

```javascript
// 粒子參數
count:     20 個粒子同時存在
size:      4px ~ 8px 隨機
color:     #c8a84b（主金色）/ #f0d060（亮金色）交替
opacity:   0.4 ~ 0.9 隨機
duration:  3s ~ 7s 浮升動畫（CSS keyframes）
position:  x 軸隨機分布，y 軸由下往上消失
```

## 3. 登入框元件規格

### 尺寸與位置

```css
.login-box {
  width: 360px;
  padding: 40px 36px;
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  background: rgba(10, 7, 4, 0.88);
  border: 1px solid var(--border);   /* #6a4420 */
  border-radius: 4px;
}
```

### 元件清單

| 元件 | 類型 | 說明 |
|------|------|------|
| 遊戲標題 | `<h1>` | 「🏰 天堂：精煉高爽度」，金色字體 |
| 分隔線 | `<hr>` | 細金色橫線，視覺區隔 |
| 伺服器選單 | `<select>` | 下拉選單，預設「肯特城 ─ 登入伺服器 1」 |
| 帳號欄位 | `<input type="text">` | placeholder：「輸入帳號名稱」 |
| 密碼欄位 | `<input type="password">` | placeholder：「輸入密碼」，顯示為 ●●●● |
| 登入按鈕 | `<button>` | 全寬，金色背景，文字「進入天堂」 |

### 伺服器選項

```html
<option value="s1">肯特城 ─ 登入伺服器 1</option>
<option value="s2">吉倫登 ─ 登入伺服器 2</option>
```

## 5. 資料流設計

### 登入流程

```
使用者輸入帳號 + 密碼 + 選擇伺服器
           ↓
  點擊「進入天堂」或按 Enter
           ↓
  驗證：帳號不得為空（前端驗證）
           ↓
  查詢 localStorage['lineage_player']
      ┌────────────────┬──────────────────┐
    有記錄               無記錄（新玩家）
      ↓                       ↓
  讀取現有資料          建立新玩家資料
  更新 server           初始金幣：10,000,000
  更新 loginTime        server：選定伺服器
      └────────────────┴──────────────────┘
           ↓
  location.href = 'lineage-refine.html'
```

### localStorage 結構

```json
{
  "lineage_player": {
    "name": "玩家帳號",
    "gold": 10000000,
    "loggedIn": true,
    "server": "s1",
    "loginTime": 1718880000000
  }
}
```

## 6. 互動規格

| 互動 | 行為 |
|------|------|
| `Enter` 鍵 | 觸發登入，等同點擊「進入天堂」按鈕 |
| 空帳號送出 | 顯示提示訊息（前端驗證），不跳轉 |
| 登入成功 | 立即跳轉至 `lineage-refine.html`，無延遲 |
| 頁面重整 | 若已有 `loggedIn` 記錄，可考慮自動跳過登入（未來功能） |

## 7. 轉場設計

登入成功後直接使用 `location.href` 跳轉，主遊戲頁面載入時：

1. 讀取 `lineage_player`、`lineage_inventory`、`lineage_config`
2. 若無背包資料，自動初始化 100 格空背包
3. 若無配置資料，使用預設牌價 + 預設匯率
4. 顯示 NPC 開場對話，引導玩家進行第一次精煉
